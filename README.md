# Usage
Use `python` to compile the application. `togore` uses the `argparse` library, so make sure to add the necessary arguments. Yeah...
## Arguments
| Flag | Description              |
|------|--------------------------|
| -s   | Work seconds             |
| -m   | Work minutes             |
| -r   | Work hours               |
| -S   | Rest seconds             |
| -M   | Rest minutes             |
| -R   | Rest hours               |
| -c   | Number of cycles         |
| -t   | Title of task (required) |
## Example usage
```
python togore.py -m 25 -M 5 -t "Cry" -c 5
```
